# Claude Code ルール作成ガイド

.claude/rules/ ディレクトリにモジュール化されたルールを作成する。

## 作成前の確認事項

1. **スコープ**: グローバル（全ファイル）かパス固有か？
2. **配置場所**: .claude/rules/（プロジェクト）か ~/.claude/rules/（個人）か？
3. **カテゴリ**: コードスタイル、テスト、セキュリティ、API等

## ファイル構造

- **グローバルルール**: フロントマターなし
- **パス固有ルール**: YAMLフロントマターにglobパターン指定

## 必須事項

- 1トピック1ファイルで焦点を絞る
- 説明的なファイル名を使用
- 既存ルールとの重複を確認
- CLAUDE.mdとの重複を防止
