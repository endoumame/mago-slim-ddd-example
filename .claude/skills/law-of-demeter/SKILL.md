# Law of Demeter

直接の友人とだけ話せ。見知らぬ者に話しかけるな。

## 4つのルール

メソッド M が呼び出してよいのは：
1. 自身（this/self）のメソッド
2. M の引数として渡されたオブジェクトのメソッド
3. M 内で生成したオブジェクトのメソッド
4. 自身のインスタンス変数のメソッド

**禁止**: 上記メソッド呼び出しの戻り値のメソッドを呼び出すこと（友人の友人）

## 変換パターン

1. 委譲メソッドの導入
2. 目的に応じたメソッド名
3. パラメータとして渡す
4. 振る舞いをオブジェクトに移動

## 例外（違反ではないケース）

1. 流暢API / ビルダーパターン
2. データ構造（DTO / Value Object）
3. ストリーム / コレクション操作
4. 標準ライブラリの連鎖
