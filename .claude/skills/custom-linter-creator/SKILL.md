# Custom Linter Creator

既存のリンターエコシステムを活用し、AI向けのカスタムリントルールを作成する。

## ワークフロー

1. 対象言語の特定
2. lints/ ディレクトリの初期化
3. カスタムルールの実装
4. リンター設定への統合
5. テスト違反による検証

## エラーメッセージ形式

AI向けのエラーメッセージには以下を含める：
1. 違反の説明
2. 修正手順（番号付き）
3. コンテキスト（ファイル位置と識別子）
4. スコープ（変更の範囲）
5. 理由（ルールが存在する理由）
